# C++ Weather Engine

High-performance weather data processing library.

## Building

```bash
mkdir build && cd build
cmake ..
cmake --build .
```

Requires: C++17 compatible compiler and CMake 3.10+

## API

- Temperature conversions (°C ↔ °F)
- Wind chill / Heat index calculation
- Dew point calculation
- UV index classification
- Air Quality Index labeling
- Wind direction from degrees
- 7-day forecast generation
- Unit conversions (km/h↔mph, hPa↔inHg, mm↔inches)
- Seasonal weather safety tips
