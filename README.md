# 🌤️ WeatherView

**Beautiful weather forecasts — free for everyone.**

WeatherView is a daily weather app built with Python, JavaScript, and C++.
Works on **PC, tablet, and mobile** with a beautiful glassmorphism UI and smooth animations.

---

## 🚀 Access the App

| Platform | How to Access |
|----------|--------------|
| 🖥️ **PC / Desktop** | [**Open Weather App**](/) — works in any browser |
| 📱 **Mobile / Phone** | **[weatherview.app/download](/download)** — add to home screen |
| 📦 **Source Code** | [github.com/Joydip-hub/weather](https://github.com/Joydip-hub/weather) |

### ⚡ Quick Install (Mobile)

**Android:** Open in Chrome → ⋮ Menu → **Add to Home Screen**
**iPhone:** Open in Safari → Share → **Add to Home Screen**

No app store needed. Works offline after install.

---

## ✨ Features

- **20+ cities** worldwide with realistic weather data
- **7-day forecast** with temperature chart
- **24-hour hourly** breakdown
- **Dark/Light mode** toggle
- **PWA support** — install like a native app
- **Keyboard shortcuts** — Ctrl+K search, Ctrl+R refresh, Ctrl+D dark mode
- **Desktop app** — standalone executable (Windows, macOS, Linux)
